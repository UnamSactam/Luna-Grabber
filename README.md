## Features

- Discord Info
    - Token
    - Nitro
    - Billing
    - 2FA 
    - Email
    - Phone
    - Gift Codes
    - Backup Codes

- Browser Data
    - Cookies
    - Passwords
    - History
    - Credit/Debit Cards

- Game Data
	- Minecraft Session Info
	- Roblox Cookie & Other Data

- Discord Injection
    - Sends token, password & email on user login or when user changes password

- System info
    - User
    - OS
    - System
    - Network IP
    - Country
    - Wifi
    - Mac
    - Hwid
    - PC Specs
    - Screenshot
    - Proxies
    - Installed AV's
    - Product Key

- General Functions
    - Checks if being run in a virustotal sandbox/virtual machine
    - Checks for blacklisted users, pc names, HWIDs, IPs, MACs and Processes
    - Adds file to startup
    - Anti Spam
    - Fake Error
    - Obfuscation
    - Icon
    - Low Detections
    - Bypass Token Protector
    - File Pumper
    - Self Destruct
    - Clipboard
    - Crypto Wallets
